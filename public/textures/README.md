# Earth Textures

Place your Earth texture file here:

## Required Files

### Main Texture
- `earth-placeholder.jpg` - A placeholder Earth texture (2048x1024 minimum)
- `earth-8k.jpg` - High quality Earth texture (8192x4096 recommended)

### Tile-based Textures (Optional)
For better performance with large textures, use tiles:
- `tiles/earth-tile-{x}-{y}.png` - where x is 0-7 and y is 0-3

## Recommended Sources

Free Earth textures:
1. NASA Visible Earth: https://visibleearth.nasa.gov/
2. Solar System Scope: https://www.solarsystemscope.com/textures/

## Texture Requirements
- Format: JPG or PNG
- Resolution: 8192x4096 (8K) recommended for best quality
- Projection: Equirectangular (standard lat/long mapping)
