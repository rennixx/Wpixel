# Earth Texture Required

Place an Earth texture image file named `earth-8k.jpg` in this folder.

## Where to get Earth textures:

1. Solar System Scope (Recommended)
   https://www.solarsystemscope.com/textures/
   - Download "Earth Day Map" at 8k resolution
   - Rename to `earth-8k.jpg`
   - Place in this folder

2. NASA Visible Earth
   https://visibleearth.nasa.gov/
   - Search for "Blue Marble"
   - Download high-resolution version

## Recommended settings:
- Resolution: 8192 x 4096 (8k)
- Format: JPG
- Color space: sRGB
